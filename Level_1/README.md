# data_analysis
This is level 1 Data Analyzer.

In this level it reads the data and we can ask some basic questions related to the data, it provides the desired output like:
- size of data
- data
- sort by salary, age performance score
- average salary, age, performance score
- Plots like employee salary or employee performance score

In this project we have removed all the duplicates by name and changed the null values to the average of their respective column.
